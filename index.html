clover


super simple anti-malware / malware sabotage tool

Download Source

Clover is an anti-malware software made in python that tries to exterminate malware in real time.
It can sabotage malware that has debugging detection,
monitor startup changes looking for malware traces,
monitor DNS records looking for DNS poisoning,
among other cool things. :D

goes better with common sense!
